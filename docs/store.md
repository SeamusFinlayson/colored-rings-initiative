# Rings Initiative

Use colored rings create initiative groups. This extension currently only supprts checkbox style initiave.

## How to use

Select your character or mount token and then click the colored rings icon that appears on the context menu. You can choose what color you would like your ring. Select another color to add another ring. Alternativly, right click on a token the select the "Add Initiative" option.

In the Initiative Menu you can:

- Click the lightning (triggered action) or the flag icon (turn) to use that attribute for the first item in that group
- Right click or hold the lightning or flag icon to toggle that attribute for all memeber of that group
- Click a token to enter Group View
- Right click a group to enter Group View with all tokens selected
- Click "Initiaitive" to configure the extensions's room settings
  - Broadcasting the round allows other extensions to detect when the round changes, Draw Steel Tools makes use of this event
- Click "Round X" to edit the round

In Group View you can:

- Toggle the lighting and flag icons for each token individually
- Select tokens and use the following options from the top bar:
  - Deselect all tokens
  - Focus tokens
  - Switch catagories
  - Set a custom number of triggered actions for tokens
  - Set a custom number of turns for tokens
  - Remove tokens from initiaitive
- Click the top bar to select all tokens
